ステップ3：トリガーとカスタムメニューの使い分け

あなたのツールは、トリガー設定とカスタムメニューのどちらが適しているでしょうか？

GASツールの実行方法には、大きく分けて2つの方法があります。 あなたが作りたいツールに合った方法を選ぶことで、より使いやすいツールになります。

⏰ トリガー設定

決まった時間や条件で自動実行する方法です。 一度設定すれば、人の手を介さず自動で動きます。

📋 カスタムメニュー

スプレッドシートのメニューにボタンを追加して、クリックで実行する方法です。 実行タイミングを自分で決められます。

⏰ トリガー設定が向いているツール

こんな場合に最適

✓

完全自動化したい - 人の手を介さずに処理を実行したい

✓

定期的に実行したい - 毎日、毎週、毎月など、決まったタイミングで実行したい

✓

フォーム送信時に自動実行したい - Googleフォームの回答があったら即座に処理したい

✓

人の手を介さず放置したい - 設定したら忘れていても勝手に動いてほしい

実行例

  • • 毎朝9時に注文メールを自動転記
  • • フォーム回答時に自動返信メールを送信
  • • 毎週月曜日に週報を自動作成
  • • 1時間おきに在庫状況をチェック
実践① トリガーの設定方法を確認 →

📋 カスタムメニューが向いているツール

こんな場合に最適

✓

手動で実行したい - 自分で実行タイミングをコントロールしたい

✓

実行するタイミングを自分で決めたい - 必要な時だけ実行したい

✓

複数の機能を1つのメニューにまとめたい - いくつかの処理をメニューから選びたい

✓

初心者でも簡単に使えるようにしたい - ボタンをクリックするだけで実行できる

実行例

  • • 「注文データを取り込む」ボタンをクリックして実行
  • • 「レポートを作成」ボタンをクリックして実行
  • • 「メール一斉送信」ボタンをクリックして実行
  • • 「データをクリア」ボタンをクリックして実行
実践② カスタムメニューの設定方法を確認 →

🔄 組み合わせも可能

トリガーとカスタムメニューは、**両方使うこともできます**。 どちらか一方だけを選ぶ必要はありません。

💡 組み合わせの例

1

カスタムメニューで手動実行もできる
通常は手動で実行したいけど、緊急時にすぐ実行できるようにメニューも用意しておく

2

トリガーで毎日自動実行もできる
普段はトリガーで自動実行しつつ、必要に応じて手動でも実行できるようにする

3

両方の良いとこ取り！
「自動実行＋手動実行」の両方ができる、便利なツールになります

まずはどちらか一方から始めて、後から追加することもできます。 最初から完璧を目指さず、段階的に機能を追加していくのがおすすめです。
